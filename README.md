# ceng-407-408-2025-2026-AI-Assisted-Programming-Platform-for-Education-and-Assessment

AI-Assisted Programming Platform for Education and Assessment

## Project Structure

- `frontend/` → React frontend
- `backend/` → Node.js backend
- `infra/` → DevOps scripts and infrastructure files
- `docs/` → Project documentation

## Services

- Frontend
- Backend
- PostgreSQL
- Judge0
- OpenWebUI

## Environment Setup

1. Copy `.env.example` to `.env`
2. Update values if needed

## Run Project

```bash
docker compose up --build
```

## Stop Project

```bash
docker compose down
```
