Title of the Project  AI-Assisted Programming Platform for Education and Assessment  
Project Description  
The project aims to design and implement a web -based application to support programming classes, laboratory works, and exams. Teachers can 
create and assign programming problems with specific deadlines. Students will be able to attempt these problems in a simple, browser -based IDE 
within the platform.  
 
The system’s core innovation is its AI integration:  
 
• Students cannot directly ask the AI to solve problems. Instead, they can request guidance and feedback on their code.  
• The AI will provide hints, reminders, and improvement suggestions, ensuring students learn problem -solving skills rather than copying 
solutions.  
• For teachers, AI can generate similar question variations, help design rubrics, and provide score recommendation s by analyzing 
students’ submissions.  
 
The platform will be designed with scalability in mind, allowing integration with cloud -based or local AI servers (e.g., using Ollama and different 
LLMs). The project will combine frontend design, backend development,  AI prompt engineering, and educational assessment methods.
